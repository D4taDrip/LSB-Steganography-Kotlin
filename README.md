# LSB Steganography Application

## Описание
Приложение для встраивания текстовой информации в BMP изображения по алгоритму НЗБ (наименее значащий бит), написанное на Kotlin с использованием JavaFX, Gradle и других современных технологий.

## Особенности
- Графический интерфейс на JavaFX
- Логирование с использованием log4j2
- Сборка проекта через Gradle
- Unit-тесты на JUnit
- Поддержка Kotlin
- Подробная документация JavaDoc
- Использование современных принципов программирования

## Требования
- Java 18 или выше
- Gradle (для сборки)

## Установка и запуск
1. Клонируйте репозиторий: `git clone https://github.com/D4taDrip/Lsb-Steganography-Kotlin.git`
2. Перейдите в директорию проекта: `cd Lsb-Steganography-Kotlin`
3. Соберите проект: `./gradlew build`
4. Запустите приложение: `./gradlew run`

## Использование
1. Выберите BMP изображение с помощью кнопки "Обзор" в поле "Путь к исходному изображению".
2. Введите текст для встраивания в соответствующее поле.
3. Укажите путь для сохранения результата.
4. Нажмите кнопку "Встроить текст".
5. Программа отобразит оригинальное изображение, изображение с вложением и визуализацию младших битов.

## Архитектура
- `src/main/kotlin/com/example/lsbsteganography/core`: Логика обработки изображений и стеганографии
- `src/main/kotlin/com/example/lsbsteganography/gui`: Графический интерфейс
- `src/main/kotlin/com/example/lsbsteganography/utils`: Вспомогательные утилиты (например, логирование)

## Тестирование
Для запуска тестов выполните: `./gradlew test`

## Генерация документации
Для генерации JavaDoc выполните: `./gradlew generateDocs`
Документация будет создана в `build/docs/javadoc`.
